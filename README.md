# ASTRA — Tech Student Team Website

> *Beyond the Stars.*

A futuristic, cinematic website for **ASTRA**, a tech student team focused on innovation through hackathons, robotics, AI/ML, and ideathons. Built with a dark Avengers-inspired aesthetic featuring liquid chrome effects, animated intros, and immersive audio-visual experiences.

---

## 🚀 Features

- **Cinematic Intro Sequence** — Fullscreen video overlay with audio unlock and smooth transition into the main content
- **Liquid Chrome / Ripple Effect** — Interactive water ripple on desktop (via jQuery Ripples)
- **Background Music** — Immersive audio with toggle control
- **Team Page** — Dedicated `team.html` with animated member cards for Team Lead, Co-Leads, and Technical Wing
- **Scroll Animations** — Reveal-on-scroll for all major sections
- **Scroll Progress Bar** — Visual indicator at the top of the page
- **Responsive Navigation** — Mobile hamburger menu with smooth transitions
- **Glassmorphism UI** — Cards with backdrop blur and neon glow accents
- **Avengers Tech Rings** — Animated rotating ring decorations in the background
